操作

伪代码

  1. FindPath(start, goal)
  2. open ← { start }, closed ← ∅
  3. while open is not empty
  4. current ← node in open with lowest F
  5. if current == goal: return ReconstructPath
  6. add current to closed
  7. for each neighbor in GetAllNeighbours(current)
  8. if neighbor invalid, blocked, or closed: continue
  9. tentativeG ← G[current] + moveCost
  10. if tentativeG improves neighbor
  11. cameFrom[neighbor] ← current
  12. F ← G + HexDistance(neighbor, goal); add/update open
  13. return empty path
